# Thyroid Disease Data Analysis & Visualization

## Overview
This project focuses on analyzing thyroid patient data to understand hormone level patterns and relationships.
The dataset includes medical attributes such as TSH, T3, TT4, T4U, and FTI.

## Objectives
* Clean and preprocess the dataset
* Analyze key hormone indicators
* Visualize patterns and relationships
* Identify outliers and distributions

## Dataset
The dataset is sourced from Kaggle and contains thousands of patient records with multiple medical features.

## Technologies Used
* Python
* Pandas
* Matplotlib
* Seaborn

## Analysis Performed
* Data cleaning (handling missing values and incorrect formats)
* Visualization of FTI levels across patients
* Distribution analysis using histograms
* Relationship analysis between age and FTI
* Correlation analysis using heatmaps
* Outlier detection using boxplots

## Key Insights
* Most FTI values fall within a moderate range
* Presence of outliers suggests abnormal thyroid conditions
* Certain hormone levels show correlation (e.g., TT4 and FTI)
* No strong relationship observed between age and FTI

## Conclusion
This project provides a foundational understanding of thyroid hormone data and demonstrates basic data analysis and visualization techniques.
